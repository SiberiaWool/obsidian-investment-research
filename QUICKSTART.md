# 快速开始指南

## 安装到Obsidian

### 方法1：手动安装（推荐用于开发）

1. **构建插件**
   ```bash
   cd /home/obsidian-investment-research
   npm run build
   ```

2. **复制到Obsidian插件目录**
   - Windows: `%APPDATA%/Obsidian/plugins/`
   - Mac: `~/Library/Application Support/obsidian/Plugins/`
   - Linux: `~/.config/obsidian/plugins/`

   创建插件目录：
   ```bash
   mkdir -p ~/.config/obsidian/plugins/obsidian-investment-research
   cp -r /home/obsidian-investment-research/* ~/.config/obsidian/plugins/obsidian-investment-research/
   ```

3. **在Obsidian中启用**
   - 打开Obsidian设置
   - 进入"社区插件"
   - 关闭"安全模式"（如果开启）
   - 在"已安装插件"中找到"Investment Research"
   - 点击启用

### 方法2：开发模式（热重载）

1. **启动开发模式**
   ```bash
   cd /home/obsidian-investment-research
   npm run dev
   ```

2. **创建符号链接**
   ```bash
   # Linux/Mac
   ln -s /home/obsidian-investment-research ~/.config/obsidian/plugins/obsidian-investment-research
   ```

3. **在Obsidian中重新加载插件**
   - 按键：Ctrl/Cmd + R
   - 或在设置中禁用再启用插件

## 首次使用

### 步骤1：初始化知识库

1. 打开命令面板：`Ctrl/Cmd + P`
2. 输入：`初始化投研知识库`
3. 按回车执行

插件会自动创建以下文件夹结构：
```
投研知识库/
├── 公司研究/
├── 行业研究/
└── 概念主题/
```

### 步骤2：创建第一个公司研究

1. 打开命令面板：`Ctrl/Cmd + P`
2. 输入：`创建公司研究文件夹`
3. 按回车，弹出对话框

4. 填写信息：
   - **股票代码**：600519
   - **公司名称**：贵州茅台
   - **所属行业**：白酒

5. 点击"创建"

插件会自动创建：
```
公司研究/
└── 600519_贵州茅台/
    ├── 关键驱动因素跟踪.md       # 核心驱动因素跟踪
    ├── 市场核心分歧点.md         # 多空观点整理
    ├── 研究任务清单.md           # 研究任务清单
    ├── 相关文档链接.md           # 文档链接
    ├── 调研纪要/                 # 调研纪要文件夹
    ├── 公告资料/                 # 公告资料文件夹
    └── 财务数据/                 # 财务数据文件夹
```

### 步骤3：开始研究

1. 打开 `关键驱动因素跟踪.md`
2. 填写核心驱动因素：
   - 驱动因素：产品提价
   - 当前状态：飞天提价10%
   - 历史变化：2026-01-01
   - 影响程度：极高

3. 打开 `市场核心分歧点.md`
4. 填写市场分歧点：
   - 分歧点：2026年能否再次提价？
   - 看多观点：需求旺盛，批价持续高位
   - 看空观点：宏观消费降级
   - 我们的观点：置信度高

5. 打开 `研究任务清单.md`
6. 基于分歧点添加研究任务：
   - [ ] 跟进Q2批价变化（截止：2026-05-30）
   - [ ] 访谈经销商3-5家（截止：2026-05-15）

## 自动功能

### 1. 自动元数据
新建文档时，插件会自动：
- 添加创建时间
- 添加修改时间
- 添加作者信息（可在设置中自定义）
- 提取标签（公司、行业、主题）
- 提取关联公司
- 识别所属行业

### 2. 自动标签
插件会自动识别：
- **公司标签**：贵州茅台、五粮液等
- **行业标签**：白酒、半导体、新能源等
- **主题标签**：提价、扩产、并购、业绩等

### 3. 自动关联
创建文档后，运行"查找相关文档"会自动：
- 查找同公司的其他文档
- 查找同行业的其他文档
- 查找相同主题的文档
- 在文档末尾添加相关文档链接

## 实用技巧

### 技巧1：快速创建调研纪要
1. 命令面板 → "创建调研纪要"
2. 插件自动创建带元数据的模板
3. 填写调研内容
4. 插件自动提取标签和关联公司

### 技巧2：跟踪研究进度
1. 在 `研究任务清单.md` 中添加任务
2. 使用 `[ ]` 和 `[x]` 标记完成状态
3. 添加截止日期：`（截止：2026-05-15）`
4. 定期回顾更新

### 技巧3：更新关键驱动因素
1. 打开 `关键驱动因素跟踪.md`
2. 在表格中添加新的驱动因素
3. 更新现有因素的状态
4. 在"更新来源"中添加信息来源

### 技巧4：利用双向链接
- 使用 `[[文档名]]` 创建文档间的链接
- Obsidian会自动显示反向链接
- 帮助发现知识关联

## 配置建议

### 基础配置
在 设置 → Investment Research插件 中：

1. **文件夹结构**
   - 知识库根路径：投研知识库
   - 公司研究路径：投研知识库/公司研究

2. **文档设置**
   - 自动添加元数据（推荐开启）
   - 自动提取标签（推荐开启）
   - 自动关联文档（推荐开启）

3. **用户信息**
   - 分析师姓名：您的名字
   - 这个名字会用于所有新建文档的author字段

## 故障排除

### 问题1：插件未显示
**解决方案**：
- 检查是否在"已安装插件"中启用
- 重新加载Obsidian（Ctrl/Cmd + R）
- 检查开发者控制台是否有错误

### 问题2：文件夹创建失败
**解决方案**：
- 检查是否有权限创建文件夹
- 确保路径不包含特殊字符
- 手动创建根文件夹后再试

### 问题3：元数据未自动添加
**解决方案**：
- 检查设置中"自动添加元数据"是否开启
- 等待1-2秒（插件有延迟）
- 手动运行"更新当前文件标签"命令

### 问题4：标签提取不准确
**解决方案**：
- 这是已知限制，MVP版本只支持关键词匹配
- 手动在元数据中添加标签
- 未来版本会集成AI进行智能提取
