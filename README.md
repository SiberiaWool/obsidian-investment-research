## 核心功能

### 1. 复盘管理

#### 日度复盘
- 使用模板：`日复盘 - Week.md`
- 自动保存到：`0.周期复盘/0.日度复盘/`
- 自动添加日期和元数据

#### 周度复盘
- 使用模板：`周复盘.md`
- 自动保存到：`0.周期复盘/1.周度复盘/`
- 自动添加周次和元数据

### 2. 个股研究

#### 分类管理
个股研究按关注程度分为三类：

1. **近期核心** - `1.研究/1.个股/0.近期核心/`
   - 当前重点研究的股票
   - 需要密切跟踪

2. **长期跟踪** - `1.研究/1.个股/1.长期跟踪/`
   - 长期关注的优质标的
   - 定期更新研究

3. **其他覆盖** - `1.研究/1.个股/2.其他覆盖/`
   - 一般覆盖的股票
   - 不定期更新

#### 自动生成研究框架
每创建一个股票研究，自动生成：
- **关键驱动因素跟踪.md** - 核心驱动因素跟踪表格
- **市场核心分歧点.md** - 多空观点整理
- **研究任务清单.md** - 研究任务列表
- **相关文档链接.md** - 自动关联文档
- **调研纪要/** - 调研纪要文件夹
- **公告资料/** - 公告资料文件夹
- **财务数据/** - 财务数据文件夹

_内容需用户手动更新_

### 3. 板块研究

创建板块研究时，自动生成：
- **板块概览.md** - 板块基本情况
- **核心个股.md** - 板块内重点公司
- **市场观点.md** - 市场对板块的看法
- **跟踪指标.md** - 需要跟踪的指标

保存到：`1.研究/0.板块/{板块名称}/`

### 4. 智能文档关联

#### 自动关联功能
- **公司关联**：同一公司的所有文档自动关联
- **板块关联**：同一板块的文档自动关联
- **标签关联**：相同标签的文档自动关联
- **复盘关联**：复盘中的提到的股票自动链接到研究文档

#### 关联方式
使用 `[[文档名]]` 创建双向链接，Obsidian会自动显示反向链接。

## 使用指南

### 首次使用

1. **安装插件**
   - 创建一个新的Obsidian仓库
   - 设置中启用第三方插件
   - 将本插件文件夹/obsidian-investment-research 复制到/.obsidian/.plugins 文件夹下
   - Obsidian插件设置中刷新并启用插件

2. **初始化文件夹**
   - 使用快捷键CTRL+P打开命令面板，使用命令`初始化投研知识库`初始化文件夹系统

3. **初始化模板**
   - 将/obsidian-investment-research/templates 文件夹放到和/0.周期笔记 等同级文件夹下

### 创建文档命令

#### 复盘类
1. **创建日度复盘**
   - 命令：`创建日度复盘`
   - 选择日期
   - 自动使用模板并保存到对应文件夹

2. **创建周度复盘**
   - 命令：`创建周度复盘`
   - 选择周次
   - 自动使用模板并保存到对应文件夹

#### 个股研究类
1. **创建个股研究（近期核心）**
   - 命令：`创建个股研究（近期核心）`
   - 输入股票代码、公司名称、板块
   - 保存到：`1.研究/1.个股/0.近期核心/`

2. **创建个股研究（长期跟踪）**
   - 保存到：`1.研究/1.个股/1.长期跟踪/`

3. **创建个股研究（其他覆盖）**
   - 保存到：`1.研究/1.个股/2.其他覆盖/`

#### 板块研究类
1. **创建板块研究**
   - 命令：`创建板块研究`
   - 输入板块名称（如：白酒、半导体）
   - 自动创建板块文件夹和基础文档
   - 保存到：`1.研究/0.板块/{板块名称}/`

#### 通用功能
1. **更新当前文件标签**
   - 自动提取公司、板块、标签
   - 更新文档元数据

2. **查找相关文档**
   - 自动查找并关联相关文档

3. **创建关联链接**
   - 在当前文档末尾添加关联文档链接

4. **重新加载模板**
   - 清除模板缓存
   - 修改模板后使用此命令重新加载

## 智能标签系统

### 同花顺概念板块识别

插件会自动识别以下同花顺概念板块：

**产业链类：**
- AI产业链、新能源汽车产业链、锂电池产业链、光伏产业链等

**主题类：**
- 新能源汽车、光伏、风电、电子、计算机等

### 标签层级

```
板块/{板块名称}        # 如：板块/白酒
公司/{公司名称}        # 如：公司/贵州茅台
行业/{行业名称}        # 如：行业/食品饮料
主题/{主题名称}        # 如：主题/提价
```

## 配置说明

### 文件夹结构设置

在 插件设置 → 文件夹结构 中可配置：

- **根路径**：默认为您仓库的根目录
- **日度复盘路径**：默认 `0.周期复盘/0.日度复盘`
- **周度复盘路径**：默认 `0.周期复盘/1.周度复盘`
- **板块研究路径**：默认 `1.研究/0.板块`
- **个股研究路径**：根据分类（近期核心/长期跟踪/其他覆盖）
- **模板文件夹路径**：默认 `templates`

**注意**：所有路径都是相对于仓库根目录的相对路径。

### 文档设置

- **自动添加元数据**：新建文档自动添加元数据
- **自动提取标签**：自动提取板块、公司、标签
- **自动关联文档**：自动建立文档间链接

### 用户信息

- **分析师姓名**：用于文档author字段，可在设置中自定义

## 常见问题

### Q1：我想使用现有的文件夹结构，怎么办？
**A**：在插件设置中修改各个路径为您现有的路径，所有路径都支持相对路径配置。插件会自动适配您现有的文件夹结构。

### Q2：如何修改复盘模板？
**A**：直接编辑 `templates/` 文件夹中的模板文件，修改后执行 `重新加载模板` 命令。

### Q3：如何在不同分类间移动个股研究？
**A**：当前版本需要在文件管理器中手动移动文件夹。移动后记得在插件设置中确认路径配置。

### Q4：如何备份我的模板？
**A**：`templates/` 文件夹中的所有文件都是普通的Markdown文件，可以直接复制备份。

### Q5：文档关联会不会影响性能？
**A**：文档关联只在您主动触发时执行，不会在后台自动运行，不会影响性能。
